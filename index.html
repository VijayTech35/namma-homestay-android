Everything You Need

Built for travelers, hosts, and admins — all in one app

🔍

Browse & Search

Explore verified homestays with real-time search, chip filters (All, Available, Verified, Low Price), and advanced filter dialog — 300ms debounced search for instant results.

⭐

Reviews & Ratings

Read guest reviews with star ratings. Rate your stay (1-5 stars) and write a review. Average rating calculated automatically. All reviews displayed in a dedicated section.

RatingBar RecyclerView
📅

Booking System

Select check-in/check-out dates with pickers, specify guest count, send booking requests. Host receives notifications. Booking status tracking (pending/confirmed/cancelled).

DatePicker Status Tracking
❤️

Favorites & Wishlist

Save homestays with one tap. Heart toggle with filled/outline icons. Persistent across sessions — your wishlist is always available.

📱

Share & Connect

Share homestay details via system share sheet. Open location directly in Google Maps. Call or WhatsApp the host instantly from the detail screen.

Share Intent Maps Deep Link WhatsApp
🌐

Multi-Language

Full support for English, Hindi (हिन्दी), and Kannada (ಕನ್ನಡ). App auto-detects device locale. All UI strings translated including auth, browsing, and settings.

Hindi Kannada Locale Auto-Detect
🏠

Host Dashboard

Manage your listings with quick action cards: View Inquiries, Daily Menu, Add Places. Edit or delete your homestays with confirmation dialogs.

🍽️

AI-Powered Menu

Generate daily menus with Gemini AI. Traditional Karnataka dishes auto-suggested. Edit and publish breakfast, lunch, and dinner with descriptions.

Gemini AI Auto-Suggest
🗺️

Nearby Guide Places

Add trekking spots, temples, waterfalls with categories. Guests see them on the detail screen. Tap to open location in Google Maps with directions.

Categories Map Direction
📩

Guest Inquiries

Real-time inquiry management. View messages from guests. Call or WhatsApp guests directly from the inquiry screen.

📸

Image Upload

Pick from gallery or camera. Auto-compressed to <500KB. Upload to Firebase Storage. Preview before submitting. Dashed border add-photo UI.

🛡️

Admin Panel

Verify, reject, or delete listings. Filter by All/Verified/Pending. Chip-based filtering. Swipe to refresh. Delete confirmation dialog prevents accidents.

👤

Phone Auth + Google Sign-In

OTP-based phone authentication with auto-retry. Google Sign-In with one tap. Secure session persisted via EncryptedSharedPreferences. Role selection (Guest/Host) on first login.

Phone OTP Google Sign-In Encrypted Session
🌙

Dark Mode

Full dark theme with brand colors. Toggle in Settings. Preference persisted across sessions. Auto-applied with Material Components for consistent look.

⚡

App Shortcuts

Quick actions on long-press launcher icon: Browse HomeStays, Add HomeStay, View Inquiries. Android 8+ with shortcut labels and disabled messages.

Android 8+ 3 Shortcuts
🔔

Push Notifications

FCM-based notifications for new inquiries and bookings. Cloud Functions trigger push when data changes. Notification channels for Android 8+. Permission request on Android 13+.

FCM Cloud Functions Notification Channels
📍

Location Picker

Interactive Google Maps picker for setting homestay coordinates. Tap to drop a pin, coordinates displayed below map. My Location button for current position.

Maps SDK Tap to Pin
🎨

Material Design 3

Modern UI with Material 3 components, brand colors (green #2E7D32, brown #6D4C41, cream #FFF8E1). Shimmer loading skeletons, smooth transitions, chip filters.

Material 3 Shimmer Coil Images
📋

Settings & Profile

Edit name and phone. Dark mode toggle. App version display. Logout with session clear and full back-stack cleanup via popUpTo(0).

📦

Offline Support

Firestore persistence enabled with CACHE_SIZE_UNLIMITED. Data cached locally for offline viewing. Network state observer with online/offline snackbar.

🤖

CI/CD Pipeline

GitHub Actions auto-builds on every push. JDK 17, Gradle cache, google-services.json from secrets. APK uploaded as build artifact for easy download.

GitHub Actions Auto-Build
